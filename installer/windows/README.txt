To compile the installer, you mast have the following tools/packages installed:
	* The latest AMiR development environment
	* Nullsoft installer System (NSIS) -> http://nsis.sourceforge.net/Download

 -> Build the software as usual.
 -> Right click the corresponding .nsi file and choose "Compile NSIS Script" from the context menu
 -> An executable installer is created/updated inside the "output" directory
 
 !! The Debug installer runs only on systems with VisualStudio and AMiR development environment installed !!

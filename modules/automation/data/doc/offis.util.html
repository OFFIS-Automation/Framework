offis.util Python extension

The util extension provides different automation helpers

ThreadExecute class

Class to execute single methods in a different thread
Attention:Exceptions are not handled as usual, but printed instead to stderr

class methods

ThreadExecute::ThreadExecute(reference method, list args = (), dict kwargs = {})
Creates a thread execute object that calls method in a different thread.
arguments without keyword must be given as tuple (args), arguments with keywords must be given as dict (kwargs)
Example: call print("Test", 1, 2, file = sys.stderr) threaded:
threadedPrint = ThreadExecute(print, ("Test", 1, 2), {'file' : sys.stderr} )
After the construction, status and return values can be queued from the object.

ThreadExecute::finished()
Returns true if the function is finished, false otherwise

ThreadExecute::wait()
Waits until the function has ended

ThreadExecute::value()
Returns the return value of the function. Waits until the function is finished

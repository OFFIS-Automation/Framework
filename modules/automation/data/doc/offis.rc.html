offis.rc python extension

The rc extensions provides access to the loaded RC-units (a.k.a. lolecs)

Methods

getUnit(string name)
Returns an object (RcUnit) for the requested unit

RcUnit class

Class to access functions and constants of RC-units (a.k.a. lolecs)
Each RC-unit has different methods and constants. Refer to the unit documentation
